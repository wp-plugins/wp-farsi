== Description ==

= مبدل تاریخ میلادی وردپرس به خورشیدی =


#### ویژگی‌های افزونه
+ فارسی ساز در صورت نصب نبودن زبان فارسی.
+ تنها مبدل توابع مهم mysql2date modified_time
+ تبدیل اعداد انگلیسی به فارسی 
+ رفع مشکل هاست با زبان و تاریخ
+ بسیار سبک، پایدار و ایمن


> **توجه:**  این پلاگین از دو ویجت تقویم+بایگانی و پیوندیکتا خورشیدی پشتیبانی نمی‌کند، در صورت نیاز میتوانید از امکانات پلاگین‌های جلالی و پارسی‌دیت در کنار این پلاگین استفاده کنید

== Installation ==

آموزش نصب افزونه

1.  Extract wp-farsi.zip to wp-content/plugins
2.  Go to wp-admin/plugins.php then active WP-Farsi
